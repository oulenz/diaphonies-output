How is this website made?

Diaphonies is a static website. That means that it is ‘just’ a collection of html and css files, as opposed to a website which is generated dynamically by a program running on the server, like Wordpress. In this particular instance, the html pages are made with Pelican, a python program for creating static blogs.

Which Pelican theme is this?

A modified version of Flex, by Alexandre Vicenzi.
